<h1>Ionic Firebase Login Project</h1>

 ### [YouTube Demonstration](https://) *Demo not yet linked*

<h2>Description</h2>
Project consists of a simple pages that can be used to implement a login feature to any app using the Ionic Framework by utilizing Google's Firebase Authenticator. This Program includes a landing page, a login page, a sign-up page, and a reset password page. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>HTML/CSS</b> 
- <b>Ioinic Framework</b>
- <b>FireBase Authenticator</b>

<h2>Environments Used </h2>

- <b>Visual Studio Code</b>
